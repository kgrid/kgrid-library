Postman tests readme:

1. Import the environment into postman by clicking the gear icon in the top right corner of postman
2. Click "Manage Environments" in the drop-down list
3. Click the "Import" button at the bottom of the dialog box that pops up
4. Click choose files and navigate to this directory or wherever you have the enviornment saved and select the Smoketest environment
5. Click on the name "Smoketest Environment" and change the url to point to your library instance if necessary (default is localhost:8080)
5. Close out of the dialog box
6. Click the "Import" button at the top left corner of postman
7. Make sure "Import File" is elected and click the "Choose Files" button
8. Navigate to the Library_Smoke_Tests.postman_collection and select it
9. The collection should now be in the list on the left
10. Make sure the Smoketest environment is selected and run the collection
